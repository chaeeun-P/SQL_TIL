### 기술 통계와 추론 통계
#### 기술 통계 (EDA)
기술 통계 : 문자 그대로 주어진 데이터의 특성을 사실에 근거하여 설명하고 묘사
**-> 전체 데이터를 쉽고 직관적으로 파악할 수 있도록 설명하는 것**

**-> 한 학급의 통계치를 통해 학교 전체 학생의 통계치 추정**
* 편향과 분산, 확률 분포, 가설 검정과 유의도

*기술 통계가 있어야 추론 통계가 가능하고, 기술 통계가 정확할 수록 정확한 인사이트 추출*

### 모집단과 표본추출과 관련된 개념들 

`모집단`: 분석 대상의 전체 집합  
`표본`: 모집단의 부분집합 (모집단의 일부)  
-> 단순한 부분집합이 아니라, 모집단의 편향과 이상치들을 제거한 부분집합   
`전수조사`: 모집단 자료 전체 조사 및 분석 -> 자료 추출 e.g. 대통령 선거 (완전한 모집단 X 투표 안한 건 기권으로)   
`표본조사`: 표본 추출하여 모집단의 정보 추정 및 검정 e.g. 여론조사    
-> 변수 하나당 최소 30개의 관측치 필요   

**데이터가 어느 정도 이상 확보 : 모수 == 표본 통계량**  

![사진1](/images/스크린샷%202025-03-24%20172957.png "전수조사와 표본조사 개념 예시")

**표본 추출하여 진행 후 전체 프로세스 완성 시 전체 데이터 활용하여 최종적 모델 성능 확인, 예측 및 분류**  

![사진1](/images/week1_0.png "표본 추출을 활용한 데이터 과학 프로세스")

`표본 오차`: 모집단과 표본의 자연 발생적인 변동 (e.g. 우연, 표본 수 부족) -> 확률화로 해결 가능 
`비표본 오차`: 표본 오차를 제외한 변동 -> 원인 : 편향  

* 편향의 종류
![사진1](/images/week1_2.png)
**심리적인 조건에 따른 통계적 편향도 존재**
**->브래들리 효과: 자신의 생각이나 신념을 밝히기 어려워 거짓된 응답을 함**  

* 인지적 편향의 종류
  1. 확증 편향 : 알고 있는대로 선택적으로 받아들이고 임의 판단 (멋대로 필터링) -> 신뢰성 ↓
  2. 기준점 편향 : 가장 처음에 접하는 정보에 매몰
  3. 선택 지원 편향 : 의사결정을 내리는 순간 선택의 긍정적인 부분을 더 많이 생각하고 반대되는 증거는 무시 
  4. 분모 편향 : 분수 전체가 아닌 분자에만 집중 -> 현황 왜곡
     * 절대적인 수는 크지만, 비율로 따지면 더 작을 수도
  6. 생존자 편향 : 소수의 성공 사례 -> 일반화
 
 * 머신러닝 모델의 편향과 분산
   * 편향 : 예측값들과 정답과의 일정한 차이의 정도
   * 분산 : 데이터 포인트 (e.g. 평균) 에 대한 모델 예측의 가변성 (데이터 포인트 계산할 때마다 바뀌는가?)  
     **-> 편향과 분산은 트레이드오프 관계**

### 그렇다면, 표본 편향을 최소화하는 표본 추출 방법은?
* 표본추출 종류
  1. 데이터 수집 단계의 표본 추출 (설문조사 표본 구하기)
  2. 기업에서 이미 가지고 있는 빅데이터에서 모델링을 위해 표본 추출
 * 표본 추출 단계
   1. 모집단 확정
   2. 표본 프레임 결정 (조사 대상의 목록 설정)
      1. e.g. A 회사 직원을 모집단으로 확정, 현 시점의 인사 시스템에 등록된 임직원 목록
   3. 표본 추출방법 결정
      1. 확룰 표본추출방법 (대부분, 수집 데이터 < 보유 데이터에 더 많이 사용) -> 무작위로 추출될 확률을 미리 알 수 있을 때 
         * 단순 임의 추출 : 제비뽑기, 로또 당첨 번호 같은 것
           -> 모든 모집단이 표본으로 선정될 확률 동일 
         * 계층적 표본 추출 : 일련번호 부여한 뒤 일정한 간격으로 표본 선택  
           -> 공평하게 표본 추출 but 주기성 O -> 표본의 대표성 결여
         * 층화 표본 추출 : 모집단을 특정 기준으로 분류 가능할 때 각 등급에서 일정 수 표본 무작위 추출
           -> 사전 지식, 근거 필요 (편증 위험 보완)
         * 군집 표본 추출 : 모집단 특정 기준으로 분류 후, 하나의 소집단 선택해 분석  
           -> 데이터 너무 방대한 경우! 모집단 모수 반영 어려움
      2. 복원 추출 : 표본을 모수에 복원시켜 다시 추출 가능  
         -> 모집단 크기가 안 크거나, 표본이 20% 이상으로 많은 경우 
      4. 비복원 추출 : 다시 추출 불가능 -> 하나 하나 추출할 때마다 표본공간을 바꾸는 종속사건이 됨 (다음 표본의 확률에 영향)

   5. 표본 크키 결정
   6. 표본 추출

### 변수의 종류
* 양적 변수 : 수치형 변수
* 질적 변수 : 범주형 변수

* 독립 변수와 종속 변수 (서로 상관관계, 인과관계를 가짐)
  *  통제 변수 : 실험이나 설문조사 할 때 종속 변수에 영향을 주는 것
  *  독립 변수끼리 상관관계 높으면 종속 변수와의 연관성 측정 어려움 (다중공선성)  
  ![사진](/images/week1_4.png)

* 인과관계과 상관관계
  * 인과관계 : 독립 변수가 종속 변수에 **직접적** 원인
  * 상관관계 : 인과관계의 상위개념 -> 양 / 음의 관련이 있으나 꼭 **직접적** 원인은 아님 (제3자가 영향을 줬을 수도~)
    => 상관성만 있으면 상관관계!

* 기타 관계들...
  * 독립관계 : 상관계수 0
  * 의사관계 : 상관성 O, 상관성이 제3자에 의해 나타남
  * 양방향적 인과관계 : 서로 인과적 영향 (e.g. 초기투자 -> 매출액 증가 -> 후기투자 -> 매출액 증가)
  * 조절관계 : 독립변수와 종속변수 사이에 강하고 불확정적인 조절 변수가 추가됨 (조절 변수를 포함하여 관계 분석 가능)  -> 독립변수와 종속변수 간 관계의 강도 조절  
    ![사진1](/images/week1_5.png)
  * 매개관계 : 독립변수와 종속변수의 중간에 매개변수가 개입 -> 독립변수의 영향을 종속변수에 전달 (중간다리 역할) -> 독립변수의 영향 종속변수로 전달

# 04. 데이터의 기술 통계적 측정
 
```

### 중심 성향  

**데이터 분포 형태에 따라 중심성향을 나타내는 대푯값들의 위치가 달라짐**

* 평균의 종류 (이상치에 영향 많이 받음 <-> 최빈값, 중앙값) 
  * 산술 평균 : 흔한 평균
  * 가중 평균 : 가중치 더한 평균 (e.g. 모집단이 더 많을 때)
  * 기하 평균 : 시간에 따라 비율적으로 변화하는 값의 평균 (비율의 평균)  
    -> 1년차에는 2배, 2년차에는 4배
  * 조화 평균 : 기하 평균과 비슷하지만, 이건 값의 평균    
    ![사진](/images/스크린샷%202025-03-25%20001552.png)  
 
* 분산, 표준편차 : 데이터의 퍼진 정도 파악 (모집단에서는 Q, 표본에서는 S) 
  -> 분산 값을 실제 편차의 규모와 유사하게 조정한 것이 표준편차
  -> 모집단의 자유도는 N, 표본은 n-1 (표본에서 분산이 작게 계산되는 경향이 있어서)

### 산포도
* 산포도 : 대푯값을 중심으로 자료들이 흩어져 있는 정도
  * 산포도 측정 방법 : 범위 (최대-최소), 분산, 표준 편차, 사분위수 범위, 변동계수 (표준편차/산술평균 -> 서로 다른 두 자료의 산포도 비교 (평균으로 한 번 더 나누니까))
 
### 왜도와 첨도
![사진1](/images/스크린샷%202025-03-25%20003530.png)  
* 왜도 : 데이터 분포의 좌우 비대칭도 표현  
  * 피어슨의 비대칭 계수 사용
    * 정규분포 : 0
    * 왼쪽으로 치우침 (최빈값 < 중앙값 < 평균) : 양수 / 오른쪽으로 치우침 : 음수
    * 0 보다 작거나 or 3보다 크면 정규성 X -> 참고만 하기! 절대적 X  
![사진1](/images/스크린샷%202025-03-25%20003611.png)  
* 첨도 : 정규분포보다 얼마나 뾰족하거나 완만한지의 정도를 나타내는 척도
  * 기준값보다 크면 양의 첨도 (급첨, 뾰족함), 작으면 음의 첨도 (완첨, 완만함)

```
